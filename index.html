Duo-Chrome Controls

Interactive Image Composition

Image Selection

  • A - Select Image A for control
  • B - Select Image B for control
  • X - Exchange images A and B

Image Navigation

  • ← / → - Navigate through images
  • Cmd+← / Cmd+→ - Cycle colors
  • Mouse Click - Generate new pair

Size Control

  • ↑ / ↓ - Adjust active image size

Visual Effects

  • B - Toggle background color
  • C - Cycle color palettes
  • M - Cycle blend modes

Display & Feedback

  • V - Toggle visual indicators
  • I - Toggle status display

Animation & Save

  • P / Space - Pause/resume
  • S - Toggle auto-save
  • Cmd+S - Manual save

Help

  • H or ? - Toggle this help
